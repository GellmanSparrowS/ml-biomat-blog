分子动力学入门：你的第一个LAMMPS模拟

2026-06-26·2 min read·多尺度模拟

什么是分子动力学

MD通过在每个时间步求解牛顿运动方程(F=ma)来模拟原子运动。给定了初始位置、速度和力场(势能函数)，MD预测系统如何随时间演化。

对于生物材料研究，MD可以回答：

  • 胶原微纤维的杨氏模量是多少？
  • 水分子如何与丝素蛋白表面相互作用？
  • 蛋白质在拉伸载荷下会发生什么？

安装LAMMPS

LAMMPS来自Sandia国家实验室，是最广泛应用的开源MD代码。

方法一：预编译二进制（推荐新手）




sudo apt install lammps        # Linux (Ubuntu/Debian)
brew install lammps             # macOS
# Windows: https://packages.lammps.org/windows.html




方法二：Conda（跨平台）





conda create -n lammps -c conda-forge lammps -y
conda activate lammps




验证安装：





lmp_serial -h




LAMMPS输入脚本结构



一个LAMMPS输入文件有四个基本部分：






# 1. 初始化
units           real        # 能量: kcal/mol, 距离: 埃
atom_style      full        # 原子带电荷、分子ID
boundary        p p p       # 所有方向周期性边界

# 2. 系统设置
region          box block 0 30 0 30 0 30
create_box      2 box       # 2种原子类型
create_atoms    1 random 500 12345 box

# 3. 力场
pair_style      lj/cut 10.0
pair_coeff      1 1 0.155 3.166  # O-O LJ参数

# 4. 运行
velocity        all create 300.0 12345
fix             nvt all nvt temp 300.0 300.0 100.0
thermo          100
dump            traj all custom 100 dump.lammpstrj id type x y z
run             10000




第一个模拟：水盒子



用TIP3P模型模拟500个水分子在室温下：






# water.lammps
units           real; atom_style      full
bond_style      harmonic; angle_style     harmonic

read_data       water_box.data
pair_style      lj/cut/coul/long 10.0 10.0
kspace_style    pppm 1e-5

minimize        1e-4 1e-6 1000 10000     # 能量最小化
velocity        all create 300.0 12345
fix             nvt all nvt temp 300.0 300.0 100.0
run             50000                     # NVT平衡
fix             npt all npt temp 300.0 300.0 100.0 iso 1.0 1.0 1000.0
run             100000                    # NPT生产




关键参数



参数含义典型值
--------------------
timestep积分步长1.0-2.0 fs
cutoff非键作用截断距离10-12 埃
temperature系统温度300 K(室温)
NVT恒粒子数/体积/温度平衡阶段
NPT恒粒子数/压力/温度生产阶段

轨迹分析




import MDAnalysis as mda
from MDAnalysis.analysis import rdf

u = mda.Universe("water_box.data", "dump.lammpstrj")
oxygen = u.select_atoms("type O")
rdf_oo = rdf.InterRDF(oxygen, oxygen, nbins=200, range=(0, 10.0))
rdf_oo.run()

深入探讨：力场选择与验证



力场是分子动力学模拟中最重要的输入之一。力场定义了原子间相互作用的函数形式和参数，直接影响模拟结果的准确性。为你的生物材料体系选择合适的力场是模拟成败的关键决策。



对于蛋白质和生物分子体系，CHARMM和AMBER是最成熟的两个力场系列。CHARMM36m特别优化了蛋白质的构象平衡，适合模拟蛋白质折叠和构象变化。AMBER ff19SB改进了主链扭转角的描述，在蛋白质-配体相互作用方面表现优异。对于纤维类蛋白质如丝素和胶原，两个力场都适用，选择哪个主要取决于你实验室的传统和你参考的文献使用的力场。OPLS-AA在有机小分子方面表现更好。



力场验证是经常被忽视但至关重要的步骤。在进行大规模生产模拟之前，用你知道正确结果的简单系统验证力场——例如计算水的密度和扩散系数，或者计算已知实验值的短肽构象分布。如果力场无法重现这些基本性质，那么对复杂系统的模拟结果就需要谨慎解释。这是一个非常重要的科学原则——永远在你可控的范围内验证你的方法。

  • Huang, J. et al. (2017). CHARMM36m: an improved force field for folded and intrinsically disordered proteins. Nature Methods, 14, 71-73.

补充：LAMMPS模拟在生物材料研究中的具体应用



分子动力学模拟在生物材料研究中具有独特的价值。它能够在原子尺度上揭示实验手段难以直接观测的分子机制，例如beta-折叠结晶在拉伸过程中的展开和再折叠、水分子的氢键网络如何响应力学载荷、以及丝素蛋白重链与轻链之间的界面力学。这些微观洞察为理解和设计高性能生物材料提供了分子基础。



对于刚接触MD模拟的研究生，建议从经典的水盒子教程入手，熟悉LAMMPS的输入脚本结构和基本命令。掌握了基础后，可以逐步尝试更复杂的生物分子体系。开始任何新模拟前，先查阅文献了解该体系常用的力场和模拟参数——这可以节省大量试错时间。模拟结果应始终与可用的实验数据交叉验证，建立对模拟准确性的信心后再扩展到实验无法触及的领域。




从模拟到发表：MD数据的分析和可视化



分子动力学模拟产生大量的轨迹数据，有效的分析和可视化是将数据转化为科学发现的关键。径向分布函数用于表征液体和凝胶材料的结构有序性——对于丝素蛋白水溶液，水分子之间的氧氧RDF反映了水化层的结构特征。均方位移分析给出分子的扩散系数，对于了解离子在丝素蛋白膜中的传输速率很重要。氢键分析揭示beta-折叠结晶内部和表面的氢键网络如何响应温度变化和力学载荷。



可视化工具的选择影响对模拟结果的直观理解。VMD是免费且功能强大的分子可视化软件，支持多种轨迹格式并具有脚本化渲染功能。PyMOL在生成发表级分子图像方面表现出色。对于批量分析，使用MDAnalysis库编写Python脚本实现自动化的轨迹处理和统计量提取是高效的工作流程。掌握这些分析和可视化工具意味着你可以从模拟中提取定量信息并将它们呈现在论文中，而不仅仅是展示几张快照式的分子图像。



模拟结果的多尺度验证与发表指南



分子动力学模拟的结果需要与实验数据进行比较来验证其准确性。对于丝素蛋白材料，可比较的参数包括广角X射线散射得到的beta-折叠间距和晶体尺寸、差示扫描量热法得到的变性温度、以及拉伸测试得到的杨氏模量。模拟与实验在定性趋势上的一致性比定量数值的精确匹配更为重要，因为力场参数的限制和模拟时间尺度的差异使得完美的定量匹配非常罕见。



在论文中报告MD模拟方法时，应当包括以下关键信息以便审稿人和读者评估工作质量：使用的力场和版本、模拟软件和版本、系统大小和组成、平衡和生产的模拟时长、温度控制算法、压力控制方案、以及所有非默认参数的理由。不同力场对同一体系的模拟可能产生显著不同的定量结果，因此报告力场选择是必要的科学透明性。



模拟数据的长期存储和共享有助于促进科学的开放性。将结构文件、输入脚本和处理分析脚本一起上传到公共仓库如GitHub或Zenodo，使其他研究者能够复现你的模拟结果。这不仅符合越来越多期刊的数据共享政策要求，也有助于建立你在这一领域的学术声誉。养成将每个发表项目的数据和代码系统归档的习惯。
