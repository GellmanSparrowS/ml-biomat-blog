What is FiberNet? / 是什么

A research-grade Python toolkit providing a complete closed-loop workflow for computational design of fiber network structures:

Generation → Simulation → Feature Extraction → Machine Learning → Reinforcement Learning

BMG-FDU, Fudan University / 复旦大学 生物大分子课题组

Why FiberNet? / 为什么选择

  • One-line APIs — generate, simulate, predict, optimize in a single call
    一行代码完成生成、模拟、预测、优化
  • Taichi GPU acceleration — mass-spring dynamics with auto-relaxation
    Taichi 质点弹簧动力学，GPU 加速
  • Parametric control — 368D continuous action space for RL
    368维连续动作空间，原生RL支持
  • 94-dim features — structural, pore, and contact descriptors
    94维结构-孔隙-接触特征

One-Line API / 一行代码

import fibernet as fn

# Generate / 生成
g = fn.pattern_2d(unit="honeycomb", grid=(4,4))

# Visualize / 可视化
fn.show(g)

# Simulate / 模拟
r = fn.simulate(g, mode="stretch", strain=1.5)

# ML train / 机器学习
result = fn.predict_from_csv("data.csv", target="max_force")

# RL optimize / 强化学习
best = fn.run_bayesian_optimization(obj_fn, space, n_iter=50)

Machine Learning / 机器学习

Train predictors on 94-dimensional structural features from predict_from_csv(). Built-in nested cross-validation (no data leakage), model comparison, and visualization. Random Forest, Gradient Boosting, and Ridge regression out of the box.

94维结构特征自动提取，内置嵌套交叉验证、模型对比与可视化。

Reinforcement Learning / 强化学习

Bayesian optimization over a 368-dimensional continuous action space of internal node displacements. run_bayesian_optimization() handles the full loop: generate → simulate → evaluate → update.

贝叶斯优化遍历368维连续节点位移空间，自动完成生成→模拟→评估→更新闭环。

Simulation / 模拟

Taichi-powered mass-spring dynamics with auto-relaxation, trajectory recording, and multi-frame stress visualization. Nodes as point masses, edges as linear springs with configurable stiffness. Supports stretch, compression, and custom loading modes.

Taichi 加速质点弹簧动力学，自动弛豫、轨迹记录、多帧应力可视化。

Voronoi under 1.5x stretch / Voronoi 1.5倍拉伸
