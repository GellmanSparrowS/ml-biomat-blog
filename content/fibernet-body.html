What is FiberNet? / 是什么

A unified Python framework for computational study of fiber network structures. From random networks to hierarchical chiral architectures, woven composites, and mechanical metamaterials — one toolkit for generation, simulation, analysis, and visualization.

面向纤维网络结构计算研究的统一 Python 框架。覆盖随机网络、层级手性结构、编织复合材料和力学超材料的生成、模拟、分析与可视化全流程。

BMG-FDU, Fudan University / 复旦大学 生物大分子课题组

  • Unified API — one consistent interface across 79+ generators and 22+ simulators
    统一接口，跨越 79+ 生成器与 22+ 模拟器

Generators / 生成器

Random, ordered, hierarchical, chiral, woven, bundled, curved, fractal, gradient, laminates, and metamaterial architectures including auxetic, octet, diamond, and gyroid lattices.

随机、有序、层级、手性、编织、束状、弯曲、分形、梯度、层压，以及拉涨、八面体、金刚石、螺旋面等超材料构型。

Simulation / 模拟

FEM mechanics, nonlinear deformation, fracture and damage evolution, fatigue life prediction, creep, thermal analysis, electromagnetic response, acoustic propagation, fluid-structure interaction, rheology, DMA, viscoelasticity, and multiscale homogenization.

有限元力学、非线性变形、断裂损伤、疲劳寿命、蠕变、热分析、电磁、声学、流固耦合、流变、DMA、粘弹性、多尺度均匀化。

Analysis / 分析

Morphological characterization, topological graph analysis, spatial statistics, percolation threshold detection, stress-strain curve extraction, effective property computation, Poisson's ratio evaluation, and machine learning feature pipelines.

形态表征、拓扑图分析、空间统计、逾渗阈值、应力应变曲线、等效性质、泊松比、机器学习特征管道。

I/O & Visualization / 读写与可视化

Native support for JSON, YAML, LAMMPS data, VTK, GMSH, PDB, XYZ, HDF5, and pandas DataFrames. 3D rendering via PyVista, 2D via matplotlib, interactive via Plotly, with animation export.

原生支持 JSON、YAML、LAMMPS、VTK、GMSH、PDB、XYZ、HDF5、pandas。PyVista 三维渲染、matplotlib 二维、Plotly 交互、动画导出。

Reinforcement Learning / 强化学习

Train RL agents to design and optimize fiber network architectures. Use reinforcement learning for inverse design of metamaterials with target mechanical properties, automated topology optimization, and active learning-driven structure-property exploration. Built on stable-baselines3 and Gymnasium with custom fiber network environments.

训练强化学习智能体进行纤维网络结构设计与优化。面向目标力学性能的超材料逆向设计、自动化拓扑优化、主动学习驱动的结构-性能探索。基于 stable-baselines3 与 Gymnasium，提供自定义纤维网络环境。

Quick Start / 快速开始

from fibernet import gen
from fibernet.sim.mechanical import compute_effective_properties

# Auxetic honeycomb / 拉涨蜂窝结构
net = gen.reentrant_honeycomb_2d(
    reentrant_angle=150,
    grid_size=(8, 8),
)

# Effective properties / 等效力学性能
props = compute_effective_properties(net)
print(props.summary())
