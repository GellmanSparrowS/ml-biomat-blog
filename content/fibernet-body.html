What is FiberNet?

A research-grade Python toolkit for computational study of fiber network structures. From random networks to hierarchical chiral architectures, woven composites, and mechanical metamaterials — FiberNet provides a unified framework for generation, simulation, analysis, and visualization.

Built for materials scientists, biomechanics researchers, and computational mechanicians at BMG-FDU (Fudan University).

FiberNet 是什么？

面向纤维网络结构计算研究的 Python 工具包。覆盖从随机网络到层级手性结构、编织复合材料和力学超材料的生成、模拟、分析与可视化全流程。服务于材料科学、生物力学和计算力学研究者。

由复旦大学 BMG 课题组 开发维护。

Core Capabilities

  • 79+ Generators — Random, chiral, woven, hierarchical, auxetic, octet, gyroid
  • 22+ Simulators — FEM, nonlinear, fracture, fatigue, creep, thermal, multiscale
  • Analysis — Morphology, topology, percolation, homogenization, Poisson's ratio
  • ML — Feature extraction, dataset generation, GNN, property prediction
  • I/O — JSON, YAML, LAMMPS, VTK, GMSH, PDB, XYZ, HDF5
  • Visualization — PyVista 3D, matplotlib 2D, Plotly, animations

Quick Start

pip install git+https://github.com/GellmanSparrowS/fibernet.git

from fibernet import gen
from fibernet.sim.mechanical import compute_effective_properties

# Generate auxetic honeycomb
net = gen.reentrant_honeycomb_2d(
    reentrant_angle=150, grid_size=(8,8))
props = compute_effective_properties(net)
