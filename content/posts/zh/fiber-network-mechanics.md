## 为什么纤维网络很重要

自然界用纤维构建结构材料：肌腱中的胶原蛋白、蚕茧中的丝素蛋白、植物细胞壁中的纤维素。这些**无序纤维网络**通过架构而非化学组成实现卓越的力学性能——高比强度、应变强化和韧性。

理解纤维网络力学对设计更好的生物材料至关重要。

## 关键力学行为

### 1. 应变强化

与金属（屈服后软化）不同，纤维网络**越拉越硬**。低应变时，纤维重新取向排列。高应变时，已排列的纤维轴向承载：

```python
import numpy as np
# 简化模型：纤维网络的应力-应变
strain = np.linspace(0, 0.5, 100)
stress = 0.1 * strain + 5.0 * strain**3  # 低应变弯曲主导，高应变拉伸主导
```

### 2. 架构-性能关系

| 参数 | 对力学性能的影响 |
|------|----------------|
| 纤维直径 | 更细 = 网络更柔 |
| 网络密度 (mg/cm3) | 更高 = 模量更高 |
| 交联密度 | 更多交联 = 更刚，韧性下降 |
| 纤维取向 | 取向排列 = 各向异性 |
| 纤维长径比 | 更长 = 更好的载荷传递 |

### 3. 逾渗阈值

纤维网络需要最低密度才能传递载荷——**逾渗阈值**。低于此密度，网络是互不连接的纤维"汤"。高于阈值，形成连续的载荷传递路径。

## 实验表征

常用的纤维网络力学探测方法：

- **AFM纳米压痕**：纳米级局部刚度
- **拉伸测试**：宏观应力-应变曲线
- **流变学**：粘弹性（储能模量G'、损耗模量G"）
- **SEM/TEM成像**：网络架构可视化
- **SAXS/WAXS**：变形过程中的分子级结构变化

## 计算模拟

三种主要方法：

1. **分子动力学（MD）**：原子分辨率，纳秒尺度
2. **粗粒化模型（CG）**：介观尺度，微秒尺度
3. **有限元模型（FE）**：连续介质尺度，将纤维视为梁单元

多尺度方法结合三者，从分子参数预测宏观性能。

## 常见生物材料纤维体系

- **丝素蛋白**：优异的韧性，可编程降解
- **胶原蛋白**：层级结构，细胞指导性
- **纤维素纳米纤维**：来源丰富，高刚度
- **静电纺丝聚合物**：架构可调，可规模化生产
